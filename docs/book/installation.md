# Installation

## Prerequisites

- Bun 1.3+

## Local setup

```bash
bun install
```

Run the CLI directly from source:

```bash
bun run src/cli.ts --help
```

## Create your first docs project

```bash
docia init my-docs
cd my-docs
docia dev
```

If you are running from this repository source instead of an installed binary, prefix with:

```bash
bun run src/cli.ts <command>
```
