# Team Handbook Example

This is a realistic sample docs project for `docia`.

From the repository root:

```bash
bun run src/cli.ts build --config examples/team-handbook/docia.config.ts
bun run src/cli.ts check --config examples/team-handbook/docia.config.ts
bun run src/cli.ts serve --config examples/team-handbook/docia.config.ts --build
bun run src/cli.ts dev --config examples/team-handbook/docia.config.ts
```

Generated output is written to `examples/team-handbook/dist`.
